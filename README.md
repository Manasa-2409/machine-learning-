# Random Forest Feature Importance: Interpreting Ensemble Models

## Overview
This tutorial demonstrates how Random Forest models can be interpreted using feature importance techniques. The Titanic dataset is used to identify which passenger characteristics most strongly influence survival predictions.

## Dataset
Titanic passenger dataset.

Features used include:

- Passenger class
- Sex
- Age
- Fare
- Family relationships
- Port of embarkation

Target variable: Survived.

## Tutorial Content

The notebook demonstrates:

- Data preprocessing
- Random Forest model training
- Feature importance analysis
- Permutation importance
- Comparison of importance methods
- Model evaluation
